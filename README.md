# python-team-project-pz6-vinichenko
Консольний калькулятор статистичних показників

## Мета проєкту

Розробити Python-програму для зчитування числових даних із файлу,
обчислення статистичних показників та збереження результатів у файл.

## Функціонал

- зчитування числових даних із файлу;
- обчислення середнього значення;
- обчислення медіани;
- пошук мінімального та максимального значення;
- збереження результатів у файл.

## Розподіл ролей

Студент A - зчитування даних, інтерфейс користувача, інтеграція проєкту.

Студент B - статистичні функції, перевірка помилок, тестування.

## Структура проєкту

- `main.py` - головний файл програми;
- `io_utils.py` - робота з файлами;
- `statistics_utils.py` - статистичні обчислення;
- `data.txt` - вхідні дані;
- `results.txt` - результати роботи програми.

## Запуск програми

```bash
python main.py
